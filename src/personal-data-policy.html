Политика обработки данных

1. Общие положения

Настоящая политика обработки персональных данных определяет порядок и условия обработки персональных данных, предпринимаемые компанией Trexo-Elix (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных.

Оператор ставит своей важнейшей целью соблюдение прав и свобод человека при обработке его персональных данных, включая защиту прав на неприкосновенность частной жизни.

2. Основные понятия

  • Веб-сайт — ресурс, расположенный в сети интернет по адресу Trexo-Elix.
  • Пользователь — любой посетитель веб-сайта.
  • Обработка данных — любые действия с персональными данными, включая сбор, запись, хранение, изменение и удаление.

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Персональные данные, предоставляемые Пользователем:
    • Фамилия, имя, отчество;
    • Адрес электронной почты (Email);
    • Номера телефонов.
  • Автоматический сбор: Данные файлов «cookie» и технических параметров (IP-адрес, тип браузера) с помощью сервисов интернет-статистики.

4. Цели обработки

  • Идентификация Пользователя для предоставления доступа к сервисам.
  • Установление обратной связи (уведомления, запросы).
  • Улучшение качества работы сайта и аналитика.
  • Информирование посредством рассылок (с согласия). Пользователь всегда может отказаться от рассылки, направив письмо на адрес hello@trexo-elix.blog.

5. Безопасность и хранение

Оператор принимает все необходимые организационные и технические меры для защиты данных от неправомерного доступа. Персональные данные никогда не будут переданы третьим лицам, за исключением случаев, предусмотренных законодательством.

Срок обработки персональных данных является неограниченным. Пользователь может в любой момент отозвать свое согласие, направив уведомление на email: hello@trexo-elix.blog.

6. Заключительные положения

Актуальная версия Политики находится в свободном доступе на сайте Trexo-Elix. По любым вопросам вы можете обращаться к нам по адресу: hello@trexo-elix.blog.
